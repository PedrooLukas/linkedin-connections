# Skills Aplicadas Neste Projeto

Este projeto foi construído com foco em importação, análise de dados e interface web para apoiar melhorias no perfil do LinkedIn.

## Principais skills usadas

- Java 21
- Spring Boot
- Spring Web MVC
- Spring Data JPA
- Hibernate
- SQLite
- Apache Commons CSV
- API REST
- Upload de arquivos com Multipart
- Processamento de ZIP e CSV
- Regras de negócio e análise de dados
- Testes de integração com Spring Boot Test
- Next.js
- React
- TypeScript
- Tailwind CSS
- UX/UI para dashboard analítico
- Integração frontend + backend

## Versão curta para post no LinkedIn

Neste projeto apliquei skills como:

`Java`, `Spring Boot`, `Spring Data JPA`, `Hibernate`, `SQLite`, `Apache Commons CSV`, `REST APIs`, `file upload`, `ZIP/CSV processing`, `data analysis`, `integration testing`, `Next.js`, `React`, `TypeScript`, `Tailwind CSS`, `UI for analytics dashboards` e `full-stack integration`.

## Versão mais natural para legenda

Nesse projeto trabalhei principalmente com desenvolvimento full stack, usando Java + Spring Boot no backend para importar e analisar arquivos CSV/ZIP do LinkedIn, além de Next.js, React, TypeScript e Tailwind CSS no frontend para construir a interface e apresentar os insights de forma mais clara.
